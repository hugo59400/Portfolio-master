ISLANDERS, UN JEU VIDEO EN ANGULAR

Fin : Janvier 2021

Durée : 5 mois

Technologies :

Tags :

Angular, API REST, Java, Web, Gitlab

Description :

Ce projet a été réalisé dans le cadre du cours de CPOO (Conception et Programmation Orientée Objet avancée) de 4e année en département informatique à l'Insa de Rennes. Celui-ci consistait en la conception d'un jeu vidéo.
Par binômes et à partir d'un cahier des charges pré-défini, il a fallu concevoir la structure de notre jeu (sous formes de diagrammes de classes, schémas UML, diagrammes de séquences, Mockup, ...). Le jeu a ensuite été déployé à l'aide du framework Angular pour le front-end et fonctionne avec une API REST développée en Java pour le back-end.

Bilan :

Grâce à ce projet, j'ai pu prendre en main le framework Angular plus en profondeur au travers d'un projet concret. D'un autre côté, j'ai pu mettre à profit les compétences acquises en cours de CPOO : conception logicielle, concepts avancés et patrons de conception.
