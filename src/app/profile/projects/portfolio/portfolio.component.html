Description :

Ce projet est le résultat d'un projet personnel réalisé en parallèle des mes autres projets. L'objectif était de réunir sur un site web un complément à mon CV : une description détaillée de mes projets, toutes les compétences dont je dispose, un moyen de me contacter, mais également un endroit où je partage mes reflexions du moment.
La création de ce site s'est déroulée parallèlement à la création du jeu Islanders réalisé en cours.

Bilan :

Au cours de ce projet, j'ai en parallèle créé un repository Github sur lequel j'ai pu avoir un suivi continu de mon projet. Ayant déjà utilisé Gitlab pour mes projets réalisés lors de cours, j'ai alors pu découvrir un nouvel outil utilisant Git. Au cours de ce projet, je me suis efforcé à commenter le code dès que je le pouvais, mais également à utiliser les fonctionnalités d'Issues de Github afin d'avoir un suivi complet.
