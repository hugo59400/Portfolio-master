CREATION D'UN MOTEUR DE JEU VIDEO

Fin : Juin 2021

Durée : 3 mois

Technologies :

Tags :

Ogre3D, C++, temps réel, rendu 3D, Github

Description :

Dans le cadre de cadre de mon voyage Erasmus en Suède, j'ai participé au cours 'Virtual Interactive environments' dans lequel j'ai reçu des enseignements sur la création d'un moteur de jeu vidéo. L'objectif des cours était d'acquérir des connaissances globales afin de pouvoir ensuite créer son propre moteur avec d'autres camarades.
C'est dans ce contexte que j'ai décidé de créer un moteur de jeu de combat (de type Mortal Kombat, ou Super Smash Bros) à l'aide du moteur de rendu Ogre3D et du langage C++.

Bilan :

A l'aide de ce projet, j'ai pu mieux comprendre le fonctionnement ainsi que la mise en place d'un moteur de jeu simple. Au cours de ce dernier, je me suis occupé de mettre en place un système Entity-Component permettant de simplifier grandement l'ajout de nouvelles fonctionnalités. Après ceci, j'ai donc utilisé ce nouveau système pour ajouter des éléments importants : rendu d'éléments 3D, physique,...
Pour conclure sur ce projet, nous avons rédigé un papier détaillant différentes fonctionnalités mises en place : lien vers le papier.
