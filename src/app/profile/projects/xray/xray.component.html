DETECTION DE PNEUMONIE SUR DES IMAGES X-RAY

Fin : Juin 2021

Durée : 3 mois

Technologies :

Tags :

Python, Github

Description :

Dans la cadre du cours "Advanced Deep Learning" suivi lors de mon voyage Erasmus à Luleå, j'ai participé à un projet dans lequel je devais tenter d'utiliser mes connaissances acquises en cours. L'objectif du projet était de pouvoir être capable de détecter si un patient était atteint de pneumonie ou pas à partir d'une image X-Ray de ses poumons.
Lors de ce projet, j'ai pu collaborer avec deux camarades Suédois et nous avons décidé de suivre le plan suivant : utiliser dans un premier temps un CNN puis utiliser différentes techniques de visualisation afin de mieux comprendre les données ainsi que l'apprentissage (PCA, t-SNE, visualisation de l'activation sur les images en entrée, ...), pour ensuite tenter d'effectuer la même tâche à l'aide d'une technique de transfer learning (à partir d'un modèle pré-entrainé).

Bilan :

Grâce à ce projet, j'ai pu approfondir mes connaissances en CNN, dans des techniques de Deep Learning un avancées (transfer learning, utilisation de DenseNet et d'EfficientNet) mais j'ai aussi pu comprendre l'importance de la visualisation en Deep Learning.
